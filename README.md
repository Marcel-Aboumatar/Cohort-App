## FEATURES


Website: Connect to unattired-jackson-undaughterly.ngrok-free.dev
Workflow: Go to website, log in or sign up -> if sign up then prompt for them to fill the required details, then optionally(?)  log into webadvisor 

Discover -> shows  random people in your classes, and the classes they share with you (and the ability to friend them)
Friends -> Current friends list, and viewing their profile + add/remove friends -> Friend request where you use the email of the person + dynamic list of inc friend reqs
Classes -> All the classes in your schedule (that you could import in this menu or via the import button)
^Shows the friends you got in the class and the times they happen
Calendar -> Populated calendar with details about when things happen in 24hr time
Import: Lets you do a single log into outlook to pull stuff from webadvisor

Profile page -> Information of name, email, major, bio (editable) -> make yourself undiscoverable on discovery page
^can sign out or delete the account
